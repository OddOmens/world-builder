# Fantasy World Builder

A comprehensive browser-based world-building application. Create, manage, and explore rich fictional worlds, track characters, lore, timelines, and interactive maps.

## Features
- **Entities Management**: Track characters, races, factions, items, and more.
- **Lore & Timeline**: Build intricate histories and events.
- **Interactive Maps**: Create and view maps of your world.
- **Local Storage**: Your data is stored securely in your browser using IndexedDB.

## Development Setup

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run the development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
